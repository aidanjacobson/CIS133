Pacific Trails Resort

Hands-On Practice Case

  1. The type of organization is hierarchal organization. This is the most appropriate because there is not a linear path (e.g. a slideshow) the user must take to gather relevant information from the site.
  2. Best practices:
    1. Good implementations:
      1. Informative page title that includes the company/organization/site name
      2. Main navigation links are clearly and consistently labeled
      3. All navigation hyperlinks "work" — are not broken (except reservations – to be completed later)
    2. Bad implementations:
      1. Img element tags do not use the alt attribute to configure an alternate text description
      2. Viewport meta tag is not used to enhance display on smartphones
      3. Responsive page layout is not configured for smartphone and tablet display
